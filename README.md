 ![Farmers Market Finder Demo](readmeFiles/demo.gif)


### Installation

Trillo requires [Node.js](https://nodejs.org/) v4+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ cd trillo-frontend
$ npm install
$ npm start
```
